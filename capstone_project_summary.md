
# Capstone Project Summary

## Project Title
Agentic AI for Education: Intelligent Lesson Planning and Assessment Generation

## Application 1: EduPlan AI Agent
EduPlan AI Agent is a Generative and Agentic AI-based application that helps teachers create professional lesson plans. The user provides grade level, subject, topic, lesson duration, student level, and teaching style. The AI agent generates structured learning objectives, classroom activities, assessment strategies, homework, and teacher reflection.

## Application 2: EduAssess AI Agent
EduAssess AI Agent is an AI-powered assessment generation application. The user provides grade level, subject, topic, number of questions, difficulty level, and question types. The system generates quiz or exam questions, answer keys, Bloom's Taxonomy levels, and assessment criteria.

## Why These Applications Represent Agentic AI
These applications demonstrate Agentic AI because they:
1. Accept user-defined goals.
2. Analyze the requirements.
3. Plan structured outputs.
4. Generate educational resources.
5. Support decision-making for teachers.
6. Automate repetitive academic tasks.

## Technologies Used
- Python
- Streamlit
- OpenAI API
- Prompt Engineering
- Generative AI
- Agentic AI workflow

## Educational Value
The project supports teachers by reducing planning time, improving assessment quality, and providing personalized educational content.
